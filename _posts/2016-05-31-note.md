# 程序员的技术能力模型

不要用战术上的勤奋掩盖战略上的懒惰

程序员的技术能力模型：“编程语言 30% + 抽象能力（数据结构 50% + 对现实事实的抽象理解能力 10% + 设计模式能力 10%）70% = 100%”

> * 技术价值 = 功能 + 效率 + 性能 + 稳定性
> * 产品价值 = 技术价值 * 使用量
